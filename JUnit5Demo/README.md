
# JUnit 5 Demo project

A Java project that demonstrates JUnit 5 features.

## Tech Stack

**Java:** Java 11 (Note: JUnit 5 supports Java 8 features)

**Maven:** 3.9.x

**JUnit:** 5.10.x
